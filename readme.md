# Seizedays学院 

在线教育项目

# 项目使用技术/框架

- SpringBoot
- Spring Cloud - Alibaba
- Vue
- Node.js
- Mybatis-plus
- MySql
- Swagger
- Aliyun OSS
- Aliyun Vod

# How to Use

1. 使用前请先配置好数据库 Nacos 以及Nginx
2. 后台： seizedays-edu包
3. 前台:    seizedays-edu-admin 包



# 各个端口对应功能

- qq: 2945734401
- 8110： edu-main 课程、导师等curd服务
- 8120:   sys-user  用户信息mock
- 8130: oss  对接阿里云OSS用于文件上传
- 8140： user-main  导师信息管理
- 8150: stastic 统计信息管理
- 8160: vod  对接阿里云Vod服务 用于视频的上传
